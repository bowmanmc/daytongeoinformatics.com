Dayton Geoinformatics provides expert software engineering services to those wanting to make the most of their data.

Focused on the development of web-based mapping and data visualization applications, we can help you deliver custom software applications to your users.

Web Mapping

From traditional 2D mapping with tools such as D3 and Leaflet, to WebGL-powered 3D maps using MapboxGL and Cesium, we have the expertise to build custom software that allow your users to dig deeper into their location data.

Data Visualization

Go beyond the numbers and connect the dots. Our expertise in visualizing large amounts of data can help you and your users spot trends, build stories, and gain an understanding of the meaning behind the numbers.

Web Development

We build fast, beautiful web applications while taking an agile, human-centered approach to software development. We use the latest web technologies such as Angular, React, Sass, Python, and Node.js to get the job done better and faster than our competitors.

About

Dayton Geoinformatics was started by Michael Bowman in 2017 to help companies create well designed maps, data visualizations, and custom data-centric software applications. Michael has been developing web-based software applications since 2001, working with companies such as Red Hat, Booz Allen Hamilton, and various startups in the Dayton Ohio area. In addition to his day job, Michael runs the Dayton Data Visualization meetup and volunteers with Code for Dayton, the local Code for America brigade.
